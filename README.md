# xontrib-rc-kiranmenon
my xonsh rc files

# Use cases

### Install it via pip 

```xonsh
pip install -U git+https://github.com/kiranmenon/xontrib-rc-kiranmenon 
echo 'xontrib load rc_kiranmenon' >> ~/.xonshrc
xonsh
```

### Add the RC file directly 

```bash
curl -s https://raw.githubusercontent.com/kiranmenon/xontrib-rc-kiranmenon/main/xontrib/rc_kiranmenon.xsh >> ~/.xonshrc
```
